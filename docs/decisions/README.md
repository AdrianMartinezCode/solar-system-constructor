# Decisions (ADRs, Triage, Audits)

This folder is the durable memory for how/why we changed the repo or architecture.

## Conventions

- **ADRs**: `ADR_<nnnn>_<slug>.md` (e.g. `ADR_0001_agent_roles_structure.md`)
- **Triage notes**: `TRIAGE_<slug>.md`
- **Audits**: `AUDIT_<yyyy-mm-dd>_repo_entropy.md`

## Templates

- ADR template: `ADR_TEMPLATE.md`
